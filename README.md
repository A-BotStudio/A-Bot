## Clone this project

```bash
> git clone https://github.com/A-BotStudio/A-Bot
```

```bash
>Command BOT /
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd A-Bot
> bash install.sh
```

### Usage
```bash
> npm start
```
